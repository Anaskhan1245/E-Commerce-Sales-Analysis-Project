# E-Commerce Sales Performance Analysis 🛒📊

## 📌 Project Overview
This is an end-to-end Data Analytics project aimed at analyzing the sales performance of an e-commerce business (Amazon Sales Data). The goal is to uncover business insights regarding revenue generation, product performance, regional demand, and operational efficiency.

## 🛠️ Tools & Technologies Used
* **Data Cleaning & Exploration:** Microsoft Excel
* **Database Management & Querying:** PostgreSQL
* **Data Visualization:** Python (Pandas, Matplotlib, Seaborn) in VS Code

## 📂 Files in this Repository
1. `Amazon_Sales_Analysis.sql`: Contains PostgreSQL queries used to extract insights like total revenue, top categories, and order status.
2. `sales_analysis.py`: Python script used to clean the data and generate visual charts.
3. `.png` Images: Data visualization charts generated from Python.

## 💡 Key Business Insights Discovered
1. **Top Product Category:** Analyzed which clothing categories (e.g., T-shirts, Shirts, Kurtas) generated the highest revenue, helping in future inventory planning.
2. **Regional Demand:** Identified the Top 5 states with the highest order volume to help target marketing campaigns more effectively.
3. **Operational Efficiency:** Evaluated the proportion of cancelled and returned orders to understand potential loss areas.

## 👨‍💻 How to Use
